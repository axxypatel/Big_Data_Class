Movie Recommendation System

In this notebook we are using the MovieLens 1M Dataset, we   are comparing different deep learning recommendation system models with traditional recommendation models.

Download dataset from below link:
http://files.grouplens.org/datasets/movielens/ 

Zip file to be choose: ml-1m.zip 

Unzip the file and use it for the notebook.

Algorithms implemented:
- Restricted Boltzmann Machine Algorithm
- Deep Autoencoder
- Simple Deep Neural Network
- Singular Value Decomposition
- SVD++
- Content-Based filtering
- Hybrid model

File contained in this project:

Movie_Recommendation_using_DeepLearning.ipynb
